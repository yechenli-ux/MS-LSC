# MS-LSC
Code for "A New Dual-Sieve-FFT Attack Variant Combining Modulus Switching and Lossy Source Coding".  Includes: - Small-scale experiments validating score expectation - FFT operation counts (N_FFT^(add), N_FFT^(mul)) for various p - Complexity estimation for Kyber-512/768/1024 under C0/CC/CN models  Dependencies: BLASter, G6K, FFTW.
